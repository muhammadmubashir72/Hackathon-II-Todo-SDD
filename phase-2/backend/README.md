# Pakistan Legal Assistant

This is a legal assistant chatbot specifically designed to help with questions about Pakistani laws. The assistant uses advanced AI to provide information about legal topics in Pakistan.

## Features

- Answers questions about Pakistani laws
- Explains legal concepts in simple terms
- Educational purposes only - not legal advice
- Interactive chat interface

## How to Use

Simply type your question about Pakistani law in the chat box and the assistant will respond with relevant information.

## Important Note

This system provides information for educational purposes only and should not be considered legal advice. For specific legal matters, please consult with a qualified lawyer.

## Technical Details

- Built with FastAPI and Gradio
- Powered by Google's Gemini 2.5 Flash model
- Uses the OpenAI Agents SDK

## Environment Variables

The application requires the following environment variable:
- `GEMINI_API_KEY`: Your Google Gemini API key
