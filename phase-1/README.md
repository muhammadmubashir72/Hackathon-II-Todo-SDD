# Todo CLI Application - Phase 1

This is the Phase 1 implementation of the Todo Console Application. It provides an in-memory todo list with the following features:

- Add new tasks
- View all tasks
- Update existing tasks
- Delete tasks
- Mark tasks as complete/incomplete

## Setup

1. Make sure you have Python 3.8+ installed
2. Install dependencies: `uv sync`
3. Run the application: `uv run todo-cli`

## Features

1. **Add task** - Create new todo items
2. **View tasks** - See all your tasks with status
3. **Update task** - Modify existing tasks
4. **Delete task** - Remove tasks from your list
5. **Mark complete** - Mark tasks as completed
6. **Mark incomplete** - Reopen completed tasks
7. **Exit** - Quit the application

## Usage

Run the application and follow the on-screen prompts to manage your tasks.
